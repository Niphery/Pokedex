# Pokedex Swift Application

Using: 
- Alamofire
- CollectionView
- Parsing
- SplitView
- Image Animation
- ScrollView
- SegmentView
- TableView

Known Bug: Switching the view from portrait to landscape cause an issue in the splitView. Correction in progress.
